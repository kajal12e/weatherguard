# 🌦️ WeatherGuard - Admin Dashboard

A full-stack admin dashboard for WeatherGuard with Google OAuth authentication and real-time user management.

### ✨ Live Demo
**Frontend:** https://weatherguard-xyz.vercel.app  
**Backend API:** https://weatherguard-api-xyz.up.railway.app

### 🚀 Features
- **Google OAuth Login** - Secure admin authentication flow
- **User Management** - Approve/Reject pending users from dashboard  
- **Telegram Integration** - Auto-notify users on approval via Telegram bot
- **Weather Alerts** - Trigger manual weather alerts to all approved users
- **Protected Routes** - JWT based authentication for admin panel
- **Responsive UI** - Dark theme dashboard built with React + Tailwind

### 🛠️ Tech Stack
| Area | Technology |
| --- | --- |
| **Frontend** | React 18, Vite, React Router, TailwindCSS |
| **Backend** | Node.js, Express.js |
| **Auth** | Google OAuth 2.0, JWT |
| **Database** | MongoDB Atlas |
| **Notifications** | Telegram Bot API |
| **Deployment** | Vercel, Railway |
